📋 Como conectar Google Sheets ▼
Paso 1 — Google Sheet
  1. Ve a sheets.google.com y crea una hoja nueva
  2. Fila 1 con encabezados: Fecha | Proveedor | NIT | N.Factura | Concepto | Categoria | Valor
Paso 2 — Apps Script
  1. En el Sheet: Extensiones → Apps Script
  2. Borra todo el codigo existente y pega este:
function doGet(e) {
  try {
    var p = e.parameter;
    var sheet = SpreadsheetApp
      .getActiveSpreadsheet()
      .getActiveSheet();
    sheet.appendRow([
      p.fecha, p.proveedor, p.nit,
      p.numero_factura, p.concepto,
      p.categoria, p.valor
    ]);
    return ContentService
      .createTextOutput("ok");
  } catch(err) {
    return ContentService
      .createTextOutput("error:"+err.message);
  }
}
Paso 3 — Desplegar
  1. Implementar → Nueva implementacion
  2. Tipo: Aplicacion web
  3. Ejecutar como: Yo
  4. Acceso: Cualquier persona
  5. Clic en Implementar → autoriza → copia la URL
Paso 4 — Conectar
  1. Pega la URL en el campo de Config arriba
  2. Toca el icono azul para sincronizar registros
